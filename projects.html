Sewing Pattern Maker

A web app that helps users generate sewing patterns
Personal Project, 2020-2021
  • Our web app will allow users to generate a sewing pattern based on a picture of their favorite article of clothing.
  • A user uploads a picture of a garment laying flat on a surface. Then, the user traces the seams, folds and borders of the garment using our integrated drawing tools, as well as providing a measurement. Finally, our app will automatically generate a printable pdf pattern that will allow the user to sew a copy of the garment.

Techs: React · Typescript · HTML Canvas

SafeWalk Boston

Find the safest walking route to your destination with our web app!
TechTogether Hackathon, 2019
  • Created an algorithm that computes the safest walking route between two locations according to a crime statistics database.
  • After users input an origin and destination, our website uses the Google Maps API to generate various walking routes. Our algorithm then evaluates their safety levels by taking into account the crime statistics in the locations along them.

Awards: Best Use of Open Source Data For Social Good · Climate Change Environmental Sustainability Challenge

Techs: Python · Google Maps API · HTML · GitHub Pages
